# AI Resume Analyzer

AI Resume Analyzer is a Python-based web application built using Streamlit. It analyzes PDF resumes, extracts technical skills, compares them with job descriptions, calculates a match score, and identifies missing skills.

## Features

* Upload PDF resumes
* Extract resume text
* Detect technical skills
* Compare resume skills with job requirements
* Calculate match percentage
* Identify missing skills
* Provide resume strength assessment

## Technologies Used

* Python
* Streamlit
* PyPDF2
* CSV
* Git & GitHub

## Project Structure

AI_Resume_Analyzer/

* app.py
* resume_parser.py
* skill_matcher.py
* skills.csv
* resumes/
* requirements.txt

## How to Run

```bash
streamlit run app.py
```
